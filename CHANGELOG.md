# Changelog - TrasTeoria

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
e este projeto adere ao [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.0.0] - 2026-02-17

### Adicionado

- **Plataforma Base:** Lançamento inicial com 14 seções educacionais.
- **Gamificação:** Sistema de XP, níveis, achievements e leaderboard.
- **Admin Dashboard:** Painel de administração para gerenciamento de usuários e estatísticas.
- **Novo Logo:** Integração do novo logo (Heart + Guitar).

### Corrigido

- **Segurança:** Senha do admin movida para variáveis de ambiente.
- **Deploy:** Processo de deploy estabilizado e automatizado.

### Alterado

- **Nomenclatura:** Projeto padronizado como `TrasTeoria`.
- **Versionamento:** Adotado o versionamento semântico, começando com `v1.0.0 (Maestro)`.
- **Documentação:** Consolidada e reescrita em formato profissional.
